## About this project

Echoes of the Void is a 3D action-adventure game developed in Unity. The game features procedurally generated environments, environmental storytelling, and puzzle mechanics that challenge players to think creatively.

### Key Features
- Procedural room generation system
- Environmental puzzle mechanics
- Atmospheric audio design
- Narrative-driven exploration
