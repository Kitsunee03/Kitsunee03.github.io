## About this project

Neon Drift is an arcade racing game built in Unreal Engine, featuring a striking neon visual style and tight drifting mechanics.

### Key Features
- Multiplayer racing up to 4 players  
- Dynamic track elements
- Vehicle customization system
- Synthwave soundtrack
