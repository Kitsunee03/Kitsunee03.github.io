## About this project

Pixel Quest is a web-based 2D platformer developed with Phaser.js. The game runs entirely in the browser and features a built-in level editor.

### Key Features
- Browser-based gameplay
- Built-in level editor 
- Collectible system
- Responsive controls optimized for web
